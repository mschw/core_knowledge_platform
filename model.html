Model layer¶

The model layer specifies the classes used by the Django ORM.

class core_web_service.models.Author(*args, **kwargs)¶

Represents an author in the model. Authors are not equal to Users.

class core_web_service.models.Comment(*args, **kwargs)¶

Represents a comment. Comments can be added to publications and other comments.

class core_web_service.models.Esteem(*args, **kwargs)¶

Represents the esteem a User can obtain.

class core_web_service.models.FurtherField(*args, **kwargs)¶

A key-value storage that will store values that are not part of the publication table.

class core_web_service.models.Keyword(*args, **kwargs)¶

Stores keywords for a publication that were specified by the author.

class core_web_service.models.Message(*args, **kwargs)¶

A simple message that can be sent from one user to another.

exception core_web_service.models.MissingValueException(message)¶

Raise when a required attribute is missing.

class core_web_service.models.PaperGroup(*args, **kwargs)¶

Represents a peer review group.

These groups will peer review the publications.

class core_web_service.models.PeerReview(*args, **kwargs)¶

Represents a peer review.

class core_web_service.models.PeerReviewTemplate(*args, **kwargs)¶

Represent templates for a peer review report.

class core_web_service.models.PermissionProxy(*args, **kwargs)¶

Proxy to adjust the display of permissions for the admin interface.

class core_web_service.models.ProfileField(*args, **kwargs)¶

A key-value storage for further fields of the user profile.

class core_web_service.models.Publication(*args, **kwargs)¶

Class to store publication metadata in the system.

clean()¶

Perform custom validation.

validate_required_fields()¶

Check if a publication has all fields that are required according to its type.

The required fields for a publication are based on the BibTeX standard that can be found under: BibTeX.

Arguments:
publication: the publication object to be validated.
Returns:
True when the publication fulfills the requirements.
Raises:
MissingValueException when values are missing.
class core_web_service.models.Rating(*args, **kwargs)¶

Represents a vote cast by a User for a publication of comment.

class core_web_service.models.ReferenceMaterial(*args, **kwargs)¶

Represents a reference to special material associated with a publications. This material can be any kind of file that is referenced via a url.

class core_web_service.models.ResearchArea(*args, **kwargs)¶

Store research areas for users.

class core_web_service.models.Tag(*args, **kwargs)¶

Represents a tag that can be added to a publication

class core_web_service.models.UserProfile(*args, **kwargs)¶

Stores additional information about a user not in the system.

class core_web_service.models.Vote(*args, **kwargs)¶

Represents a vote (either an up or downvote) for a comment.
